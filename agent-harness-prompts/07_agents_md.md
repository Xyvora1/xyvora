# 阶段 07 — 编写初始 AGENTS.md 引导文件

## 提示词

```
你是 AI 工程师。编写这个代码库的 AGENTS.md 文件。

该文件是 AI Agent 进入此仓库时必读的"行为宪法"。

必须包含以下章节：
1. 仓库概览（一句话描述项目）
2. 目录结构说明（每个目录的职责）
3. 禁止行为清单（例：不得修改 /domain 层的对外接口）
4. 必须遵循的编码规范（命名、注释、测试覆盖率要求）
5. 提交信息格式（Conventional Commits）
6. 如何运行本地环境
7. 如何运行测试
8. 向人类请求帮助的判断标准

语气：命令式、清晰、无歧义。
```

## 使用说明

- AGENTS.md 是整套体系中最重要的单个文件，决定 Agent 行为边界
- "命令式、无歧义"是关键——Agent 对模糊指令的处理因实现而异，要假设最保守的解读
- 第 8 条"向人类请求帮助的判断标准"尤为重要，防止 Agent 在不确定时自行假设
